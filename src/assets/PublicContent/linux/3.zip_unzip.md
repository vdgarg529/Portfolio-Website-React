## install zip

```bash
sudo apt update
sudo apt install zip
```

## zip version

```bash
zip --version
```

## zip a folder

```bash
zip -r <archive_name>.zip <folder_name>
```
-r: Recursively includes all files and subdirectories within the specified folder.
<archive_name>.zip: The desired name for your new zip file. The .zip extension is optional but recommended for clarity.
<folder_name>: The name of the folder you want to compress. 

#### Additional tags

- -e: Create a password-protected zip file
- -q: Run in quiet mode, suppressing output to the terminal
- -x: Exclude specific files or patterns (e.g., to exclude all log files)
```bash
example
zip -r project_archive.zip my_project -x "*.log"
```

## Verify the Zip File
List the contents of the archive without extracting

```bash
unzip -l project_archive.zip
```

## install unzip

```bash
sudo apt install unzip
```

## extract to current directory

```bash
unzip filename.zip
```

## Extract to a specific destination directory

```bash
unzip filename.zip -d /path/to/destination_folder
```

#### Additional tags
Common unzip options
-l: List the contents of the ZIP file without extracting them.
-P [password]: Use a password to decrypt an encrypted zip file (less secure as the password is in the command history; running the command without this flag will prompt for the password securely).
-o: Overwrite existing files without prompting for confirmation.
-q: Perform operations quietly, suppressing the normal output.
