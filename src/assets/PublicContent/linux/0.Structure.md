## Key Ubuntu Directories vs Windows

|Ubuntu Directory|Purpose (Ubuntu)|Rough Windows Equivalent|
|---|---|---|
|`/`|Root of entire filesystem|`C:\`|
|`/home`|User personal folders|`C:\Users\`|
|`/home/username`|Your files|`C:\Users\YourName\`|
|`/bin`|Essential system commands|`C:\Windows\System32\` (basic tools)|
|`/sbin`|System admin commands|Admin tools in System32|
|`/etc`|System configuration files|Windows Registry + config files|
|`/usr`|Installed applications & libraries|`C:\Program Files\`|
|`/var`|Logs, caches, variable data|`C:\ProgramData\` + logs|
|`/tmp`|Temporary files|`%TEMP%`|
|`/dev`|Hardware device interfaces|Device Manager layer (no direct equivalent)|
|`/mnt` or `/media`|Mounted drives/USBs|D:, E:, external drives|
|`/boot`|Bootloader & kernel files|Hidden boot partition|
|`/lib`|System libraries|DLLs in System32|
