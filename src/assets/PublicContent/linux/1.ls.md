## Hardware & Device Information

- **`lsblk`** — Block devices (disks, partitions)
    
- **`lsusb`** — USB devices
    
- **`lspci`** — PCI devices (GPU, network cards, etc.)
    
- **`lscpu`** — CPU architecture info
    
- **`lsmem`** — Lists memory blocks and NUMA nodes
    
- **`lsipc`** — Shows System V IPC resources
    
- **`lslocks`** — Lists current file locks
    
- **`lslogins`** — Info about system users
    
- **`lsns`** — Linux namespaces (containers, processes)


## Storage & Filesystem

- **`blkid`** — UUIDs and filesystem types of disks
    
- **`df -h`** — Disk space usage
    
- **`du -h`** — Folder sizes
    
- **`mount` / `findmnt`** — Mounted filesystems
    
- **`fdisk -l`** — Partition tables
    
- **`parted -l`** — Advanced partition info


## Memory & System

- **`free -h`** — RAM usage
    
- **`vmstat`** — Memory, CPU, IO stats
    
- **`top` / `htop`** — Live process monitoring
    
- **`uptime`** — System load and running time

## Network Hardware

- **`ip a`** — Network interfaces
    
- **`ip link`** — Network device status
    
- **`ethtool <iface>`** — NIC details
    
- **`nmcli device show`** — NetworkManager info

## All-in-One Hardware Summary (Very Useful)

These are especially helpful:

- **`lshw`** — Detailed hardware tree
    
- **`inxi -F`** — Clean full system summary
    
- **`hwinfo`** — Extensive hardware info
