## List Sessions

###### Before tmux

```bash
tmux ls
# or
tmux list-sessions
```

###### After tmux

```shortcut
ctrl + b  s
```


## New Session

### Creates a random new session with an increment session count

###### Before tmux

```bash
tmux
# or
tmux new
# or
tmux new-session
```

###### After tmux

```cmd
:new
```
### Start a new session with the name _mysession_

###### Before tmux

```bash
tmux new -s mysession
```

###### After tmux

```bash
:new -s mysession
```

# Kill a Session

## kill/delete the current session

###### After tmux

```bash
:kill-session
```

## kill/delete session _mysession_

```bash
tmux kill-ses -t mysession
#or
tmux kill-session -t mysession
```

## kill/delete all sessions but the current

```bash
tmux kill-session -a
```

## kill/delete all sessions but _mysession_

```bash
tmux kill-session -a -t mysession
```


# Start a new session or attach to an existing session named _mysession_

```bash
tmux new-session -A -s mysession
```


# Attach to last session

```bash
tmux a
#or
tmux at
#or
tmux attach
#or
tmux attach-session
```

# Attach to a session with the name _mysession_

```bash
tmux a -t mysession
#or
tmux at -t mysession
#or
tmux attach -t mysession
#or
tmux attach-session -t mysession
```

# Detach From Session

###### Detach others on the session (Maximize window by detach other clients)

```bash

```


###### After tmux

```bash
ctrl + b d
```
